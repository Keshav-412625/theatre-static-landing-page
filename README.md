# Theater Play - Floor Staff Dashboard (ENG-11594)

Yeh project Theater Floor Staff ke liye ek internal management dashboard hai. Iska uddeshya shows aur ticket logs ko handle karna aur management reporting ke liye structured data provide karna hai.

## 📋 Project Overview
Yeh ek **Static Web Application** hai jise pure HTML5, CSS3, aur Vanilla JavaScript mein banaya gaya hai. Ismein kisi bhi external framework ki dependency nahi hai, jisse yeh kisi bhi environment mein tezi se load hota hai.

## 🚀 Key Features
- **Form Management:** Naye show entries ko add karne ke liye responsive form.
- **Input Validation:** Real-time validation (agar data khali ho ya invalid entry ho, toh field red highlight hoti hai).
- **Dynamic Table Rendering:** Entry add hote hi table mein live update.
- **Empty State UI:** Jab data na ho, tab user-friendly "No data found" screen.
- **Responsive Layout:** Mobile, Tablet aur Desktop par perfectly align hota hai.

## 🛠 Tech Stack
- **HTML5:** Semantic structure ke liye.
- **CSS3:** Flexbox, Grid, aur CSS Variables (strict spacing ke liye).
- **Vanilla JavaScript:** DOM manipulation aur data handling ke liye.

## 📦 How to Run
1. Is repository ko apne system mein download/clone karein.
2. `index.html` file ko kisi bhi modern web browser (Chrome, Firefox, Edge) mein open karein.
3. Kisi server ki zaroorat nahi hai.

## 🧪 Testing/Edge Cases
- **Empty State:** Aap "Simulate Clear/Empty State" button par click karke check kar sakte hain ki jab koi record na ho toh app kaise behave karti hai.
- **Validation:** Bina koi details fill kiye "Save Entry" button dabane par validation errors check karein.
- **Responsiveness:** Browser window ko chota karke layout changes notice karein.

## 📈 Telemetry
Yeh project har interaction (log creation, initial load) ko browser console mein log karta hai taaki analytics track ki ja sake.
